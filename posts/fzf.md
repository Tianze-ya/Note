# 简介
一款模糊文件查找器
[参考](https://www.bilibili.com/video/BV1bJ411s74r)

# 下载
```bash
brew install fzf
```

# 配置
在`.zshrc`中添加`export FZF_COMPLETION_TRIGGER='\'`

# 使用
按下`\+Tab`来启动
或fzf
